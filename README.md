# OpenWaterMap

OpenWaterMap is an open-source project aimed at providing a comprehensive and user-friendly mapping solution for water bodies worldwide using advanced technologies.

## Project Structure
- `backend/`: Contains the Go backend for the application.
- `frontend/`: Contains the SvelteKit frontend.
- `docker-compose.yml`: Configuration for Docker services.
- `Dockerfile`: Instructions for building the application image.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Abuba111/OpenWaterMap.git
   cd OpenWaterMap
   ```
2. Build the backend and frontend using Docker.
3. Run the application using Docker Compose.
